# DEMO: Creating a SharePoint Framework Client-Side Web Part

This completed project is the result of the lab exercise **Creating a SharePoint Framework Client-Side Web Part** that is referenced in the [README](../../) in this repo.

## Prerequisites

- [Microsoft 365 tenant](https://developer.microsoft.com/office/dev-program?ocid=MSlearn)
- [Configured SharePoint Framework development environment](https://docs.microsoft.com/sharepoint/dev/spfx/set-up-your-development-environment)
- [Visual Studio Code](https://code.visualstudio.com/)

## Run this completed project

- Install the all dependencies:

    ```shell
    npm install
    ```

- Follow the instructions in the associated lab exercise's steps to install and test the project.
